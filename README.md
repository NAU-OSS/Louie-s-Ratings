# Louie-s-Ratings
A web application that grabs class grade distributions and evaluates trends based off teachers. A one place shop for making your next semester schedule with statistics that are publicly posted along with student sentiments for both teachers and courses.

## Installation Instructions
1. Clone into your local
2. Run `docker compose up --build`
3. Wait for docker setup to finish
4. Go to `localhost` to view the web application

## Usage Examples
- Checking Grades fro classes
- Comparing professors
- Comparing class choices based off grade distributions
